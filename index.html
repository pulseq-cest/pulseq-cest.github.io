--- layout: default title: Home --- Welcome to the Pulseq-CEST project page!

The goal of this project is that published and approved CEST saturation blocks are made available in the open standard Pulseq standard.
This allows exact comparison of CEST saturation blocks with newly developed or adapted saturation blocks for reproducible CEST research.
In addition we provide an open source Bloch-McConnell simulation which allows for simulation of the saturation blocks.


Get started with either the MATLAB or python version.
